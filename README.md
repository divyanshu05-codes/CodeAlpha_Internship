# CodeAlpha_Internship

🛒 Task 1: ShopNest E-commerce Store

ShopNest is a full-stack e-commerce web application built using the MERN stack.

Features
User Registration, Login, and Logout
JWT Authentication with HTTP-only Cookies
Persistent User Sessions
Product Listings
Product Details Page
Add to Cart
Multiple Products in Cart
Increase and Decrease Quantity
Remove Products from Cart
Cart Total Calculation
Persistent Cart using localStorage
Responsive User Interface
MongoDB Atlas Integration
Tech Stack

Frontend: React, Vite, React Router, Axios, Context API, CSS

Backend: Node.js, Express.js, MongoDB, Mongoose

Authentication: JWT, bcryptjs, HTTP-only Cookies

Tools: Git, GitHub, Postman, VS Code

🚀 Run Locally

Clone the repository:

git clone https://github.com/divyanshu05-codes/CodeAlpha_Internship.git
cd CodeAlpha_Internship

Install and start the backend:

cd Task-1-Ecommerce-Store/Backend
npm install
npm run dev

Install and start the frontend in another terminal:

cd Task-1-Ecommerce-Store/Frontend/vite-project
npm install
npm run dev
